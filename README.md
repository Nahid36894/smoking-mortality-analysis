# Smoking & Mortality Analysis — NHANES 2017-18

## Research Question
Does smoking increase mortality risk 
in US adults?

## Data
NHANES 2017-2018 + Mortality Linkage (CDC)
n = 5,401 adults aged 18+

## Methods
- Kaplan-Meier survival curve
- Cox Proportional Hazards Regression
- R packages: nhanesA, survival, survminer

## Key Findings
- Smokers have 74.8% higher mortality hazard
- Age 1 year increase → hazard 7.5% higher
- Both results statistically significant
